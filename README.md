# Isaiyana School Website

A Next.js single page school website scaffold with a `public/images` folder for your school assets.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## Project Structure

- `app/page.tsx` — main single page website with 10 sections
- `app/alumni/page.tsx` — alumni sub page
- `app/boarding/page.tsx` — boarding life sub page
- `public/images/` — store school website images here

## Adding Images

Add image files inside `public/images`, for example:
- `public/images/hero.jpg`
- `public/images/facilities.jpg`
- `public/images/team.jpg`

Then update the image `src` values in the page files.
